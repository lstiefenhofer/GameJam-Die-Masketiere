
# Masquerade at the Museum
Play as wicked thief in order to find your way through different historical exhibitions and steal ancient mask shards.

This game was developed by a team of six persons over the course of 48 hours at the Global Game Jam event in Stuttgart. All code, music, and assets were created there.

Find the game at: https://globalgamejam.org/games/2026/masquerade-museum-0
